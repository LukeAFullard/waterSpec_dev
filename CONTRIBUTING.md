# Contributing to waterSpec

We welcome contributions from the community! Whether you're fixing a bug, adding a new feature, or improving documentation, your help is appreciated.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally: `git clone https://github.com/your-username/waterSpec.git`
3.  **Create a new branch** for your changes: `git checkout -b feature/my-new-feature`
4.  **Make your changes**. Please add or update tests as appropriate.
5.  **Run the test suite** to ensure everything is still working: `pytest`
6.  **Commit your changes**: `git commit -m "Add some feature"`
7.  **Push to your branch**: `git push origin feature/my-new-feature`
8.  **Open a Pull Request** on the main `waterSpec` repository.

## Reporting Bugs

If you find a bug, please open an issue on the GitHub repository. Include a clear description of the issue, a minimal reproducible example, and information about your environment (e.g., Python version, operating system).

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms. (A `CODE_OF_CONDUCT.md` file will be added in the future).
